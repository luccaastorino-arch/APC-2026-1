Aprendizado do projeto:

1. Variáveis
Aprendi que variáveis guardam informações importantes do jogo, como score e velocidade. E devem ser as primeiras informações a serem escritas no código.

2. Sprites
Aprendi a criar personagens e objetos usando createSprite e dar animações a eles, podendo até criar a minha própria animação.

3. Funções
Aprendi a organizar o código em funções para deixar o jogo mais limpo e fácil de entender.

4. Movimentação
Aprendi a mover o jogador com teclado e objetos automaticamente com velocidade. De fato parecendo os joguinhos, os quais jogava quando era criança.

5. Colisão
Aprendi a detectar quando sprites se tocam usando isTouching.

6. Condicionais
Aprendi a usar if para mudar o jogo dependendo do score. E usar o else em determinados casos.

7. Progressão do jogo
Aprendi a criar fases diferentes mudando o background conforme o score aumenta.

8. Organização de código
Aprendi que dividir o jogo em partes (movimento, colisão, fundo, score) facilita muito o desenvolvimento, deixando o jogo mais atrativo.

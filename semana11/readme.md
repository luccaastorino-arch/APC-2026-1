Nesta semana estamos fazendo alguns exercícios, que vão ser de base para a prova, baseando na PDAD,analizamos dados das planílhas, tanto de morador quanto dos domicílios.
Começamos a fazer alguns exercícios com o professor da atividade passada no próprio github, olhando o dicionário de dados para começar, no início instalamos o pacote pandas(pd), sendo necessário instalar várais bibliotecas. O primeiro exercício foi para leras linhas e colunas, depois precisávamos guardar essas informações em uma variável.
No segundo exercício selecionamos algumas colunas em específico, por meio das listas(itens dentro das chaves).
No terceiro exercício lemos a tabela de domicílio e contabilizamos quantas pessoas moram em cada moradia,
No quarto exercício vimos o que devemos fazer para listar as idades obtidas.Finalizamos o nível 1 de dificuldade dos exercícios.
Agora no nível 2, aprendemos a usar alguns algorítimos, como por exemplo, fazer uma média aritimética, exercício feito no exercício 1, e colocamos a idade calculada em lista, e procuramos o porque não tem nenhuma idade 9999 por meio dos testes de true or false, e por fim calculamos a média, fazendo soma por soma e dividimos pelo número tota.
